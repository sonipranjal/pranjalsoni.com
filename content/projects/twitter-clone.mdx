### Idea

A web app which helps people to locate the nearby stores to a pin code.

### Tech Stack

- HTML/CSS/JS
- Node.js
- Twitter API
- Used Figma.com for UI inspiration
- Used Heroku + Netlify to deploy the app

### Features

- Beautiful UI
- Users can search tweet and find latest tweets including images/GIFs/videos
- Users can search hashtag also
