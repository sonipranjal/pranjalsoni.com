### Idea

A web app which helps people to locate the nearby stores to a pin code.

### Tech Stack

- HTML/CSS/JAVASCRIPT
- Node.js
- Google maps javascript api

### Features

Users can search pin code, and it will show the available stores near that pin code using google maps API.
