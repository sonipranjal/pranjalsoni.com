### Idea

Just a simple platform where I can host my courses and workshops.

### Tech Stack

- TypeScript
- Next.js
- Tailwind CSS
- Nest.js
- RTK Query
- MongoDB
- Prisma

### Features

- Beautiful User Interface
- Users can signup/login (http cookie based authentication with refresh token with email verification)
- Email Automation (send email to users when they create a new course OR when they register first time)
- A separate CMS for admins to manage courses and workshops
- A personalized dashboard for users to see their courses and workshops
- Course view page with user friendly navigation
- Responsive design
